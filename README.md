# Personal Portfolio Website

A modern, responsive, and full-stack personal portfolio website designed to showcase my skills, projects, certifications, education, and professional profile. The website provides an interactive user experience with a clean UI and dynamic content management.

## 🚀 Features

### Frontend
- Modern and responsive UI design
- Hero section with personal introduction
- About Me section
- Skills showcase
- Project portfolio section
- Certifications section
- Education details
- Contact form
- Mobile-friendly navigation
- Smooth animations and transitions

### Backend
- RESTful API architecture
- Contact message management
- Dynamic portfolio data handling
- Secure API communication
- Error handling and validation

### Database
- Stores:
  - Project details
  - Skills
  - Certifications
  - Contact messages
  - User/admin information

## 🔐 Authentication & Security

- JWT-based authentication
- Protected admin routes
- Secure password hashing using bcrypt
- Input validation
- Environment variable configuration

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript
- Tailwind CSS
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Tools
- Git & GitHub
- VS Code
